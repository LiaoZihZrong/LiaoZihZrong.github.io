樹莓派的OS(作業系統)

從無到有安裝整個OS

  • 使用類Linux
  • 常見的安裝方式有:
    • 直接安裝Raspberry Pi OS (previously called Raspbian)
    • 借由NOOBS安裝系統安裝Raspberry Pi OS等多種OS
      • Raspberry Pi OS與NOOBS的比較
  • 目前的Raspbian OS為32bit，雖然Pi4的硬體部份已經是64bit了

實際的安裝方式

  1. 直接從官網下載OS的image檔

    • Raspberry Pi OS的image檔
    • Ubuntu控專用的OS for Pi

  2. 把OS燒錄到記憶卡當中

    • Ubuntu系統：
      • 方式一：(使用Ubuntu內建的”製作開機碟”)

        解壓縮剛下載下來的.zip檔成.img檔
        執行”製作開機碟”
        等待15分鐘後大功告成

      • 方式二：(使用第三方軟體)

        Etcher for linux

    • win10系統：

      透過NOOBS安裝方式的教學for win10
      win32diskimager

  3. 直接透過記憶卡設定OS的wifi連線
    (1) 進入剛燒好的記憶卡之下一個名為 boot 的根目錄
    (2) 新增一個名為 wpa_supplicant.conf 的檔案，並貼上如下的設定

  4. 進入ip分享器設定Port Forwarding(NAT)：

    詳細的教程在本筆記的如下的章節有詳細說明：
    ==透過外網與Pi4溝通 (小心使用，因有資安疑慮)==

  5. Pi4的SSH預設不啟動，在 boot 的根目錄建立一個名為 ssh 的空檔案，就可以在開機時自動啟動ssh了。

    重要步驟：新建一個名為ssh空白文本，名稱後面去掉txt的後綴，注意不是新建ssh文件夾哦，特別強調是 “新建文本” 名稱是小寫的 ssh，（名稱不是ssh.txt，更不是SSH）

  6. 到此步就可以取下記憶卡，塞入Pi4，開機等待個1分鐘左右，直接以ssh連線登入Pi4

    使用ssh與Pi4溝通的方式，詳細可以先參閱後面的章節：
    ==透過SSH(secure shell)初步的建立與Pi4的溝通管道==

    ssh pi@
    (預設帳號：pi 預設密碼：raspberry)

  9. 關閉顯示器休眠，因為每次一喚醒Pi4，wifi就會死一陣子
    點擊開始（樹莓圖標）
    偏好設定

    重新啟動後即可讓螢幕不進入休眠狀態

  10. 螢幕截圖功能

    新版Raspbian裝有scrot套件，可以直接下scrot截圖、或者按Prt Scr(Print Screen)鍵
    下scrot -s指令可選定要截圖的區域(應用程式的視窗)，
    效果如同在Ubuntu、Windows底下按Alt+ Prt scn
    預設截圖儲存位置是在輸入指令的當前目錄底下
