雜亂大全12-Linux / raspberry pi install 不同版本 open model zoo (openVINO)

本篇重點:

  • 使用GitHub安裝不同版本的open model zoo

先確認自己安裝OpenVINO時的版本

以我為例 我當時下載的是2020.4版

打開open model zoo 找自己需要的模型

以我為例 我需要使用 2019 R3的版本

https://github.com/openvinotoolkit/open_model_zoo/

找到左邊的Master，點選 tag

點選 你要的版本 (ex: 2019 R3)

點選右方下載 ZIP

或是 使用指令下載

git clone -b “版本” “網址”

以上步驟就可以使用不同版本的open model zoo
預祝大家模型都能使用~
